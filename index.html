D424 Software Engineering Capstone

VacationApp is an Android mobile application designed to help users plan, organize, and manage their vacations and related activities in a simple and intuitive way. The application allows users to create vacations, track lodging information, add excursions, and receive alerts for important dates. This application was developed as a Software Engineering Capstone project, demonstrating full-stack mobile development, data persistence, testing, and cloud deployment concepts. This GitHub Pages site serves as a cloud-based deployment platform for distributing the Android application APK without requiring publication to the Google Play Store. It provides a public, versioned download location for testing and demonstration purposes.
